SP 104 - Active Learning System with Images

This project develops an Active Learning System for Chest X-Ray Analysis that enhances disease detection through intelligent data selection and model refinement. Using publicly available, privacy-compliant datasets, the system trains deep learning models to identify chest abnormalities such as pneumonia and lung lesions. Key components include dataset management, model training, active learning, evaluation, and visualization.
